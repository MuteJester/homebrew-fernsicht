# homebrew-fernsicht

Homebrew tap for [Fernsicht](https://github.com/MuteJester/Fernsicht) — real-time remote progress monitoring over peer-to-peer WebRTC.

## Install

```bash
brew tap MuteJester/fernsicht
brew install MuteJester/fernsicht/fernsicht
```

## Verify

```bash
fernsicht --version
```

## Updating

Formula files are published automatically by the
[release workflow](https://github.com/MuteJester/Fernsicht/tree/main/.github/workflows)
in the main Fernsicht repo. Don't edit `Formula/fernsicht.rb` by
hand — it's regenerated every release.
